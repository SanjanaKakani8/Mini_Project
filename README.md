# Mini_Project
MeloRhythm is a web-based music player developed using HTML, CSS, JavaScript, and Flask, featuring playlists, favorites, mini player, fullscreen player, dark mode, and music visualization.
🎵 MeloRhythm - Web-Based Music Player

## Project Overview
MeloRhythm is a modern web-based music player developed using HTML, CSS, JavaScript, and Flask. The application provides an interactive and user-friendly music listening experience with features such as playlists, favorites, recently played songs, mini player, fullscreen player, dark mode, and music visualization.

This project was developed as a Mini Project to demonstrate practical implementation of web development technologies and user interface design concepts.

## Objective
- To develop a web-based music player with a modern user interface.
- To provide seamless music playback and management features.
- To implement playlists, favorites, and recently played songs.
- To enhance user experience through responsive design and visual effects.

## Features
- 🎶 Play, Pause, Next, and Previous Controls
- 🔍 Song Search Functionality
- ❤️ Favorite Songs Management
- 🕒 Recently Played Songs
- 📂 Custom Playlist Creation and Management
- 📱 Mini Player
- 🎵 Fullscreen Music Player
- 🎨 Album Artwork Display
- 🌙 Dark Mode
- 📊 Music Visualizer
- 🔀 Shuffle Mode
- 🔁 Repeat Mode
- 💾 LocalStorage Support for Data Persistence
- ✨ Smooth Animations and Responsive User Interface

## Installation and Execution
1. Clone the repository:
git clone <repository-link>
2. Navigate to the project directory:
cd Mini\ Project
3. Install Flask:
pip install flask
4. Run the application:
python app.py
5. Open your browser and visit:
http://127.0.0.1:5000
